## Week 1

### Step 2：盤點環境、權限、資料

**環境**

- [ ] 有幾個環境？（dev / staging / production）
- [ ] 環境之間的差異？
- [ ] 網路拓撲、防火牆規則、DNS？

**權限與存取**

- [ ] 誰有 production access？存取方式？
- [ ] Secrets / credentials 怎麼管理？存在哪？
- [ ] SSH key / API token 有定期 rotate 嗎？
- [ ] 離職人員權限有收回嗎？

**資料**

- [ ] 備份策略？備份驗證過能還原嗎？
- [ ] 有 migration 機制嗎？
- [ ] 資料保留政策？合規要求？

> 驗證：能列出所有環境、存取方式、備份策略，不靠別人提示
