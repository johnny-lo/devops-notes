# 2026-03-23 Linux 學習總結

## 今日學到的指令

### 檔案列表與資訊
| 指令 | 用途 |
|------|------|
| `ls` | 列出檔案和資料夾 |
| `ls -la` | 列出全部（含隱藏檔）+ 詳細資訊 |
| `ls -R` | 遞迴列出所有子目錄內容 |

### 磁碟用量
| 指令 | 用途 |
|------|------|
| `du -sh <目標>` | 查看檔案/資料夾的大小 |
| `du -sh *` | 查看目前目錄下每個項目的大小 |
| `df -h` | 查看整個磁碟的使用量與剩餘空間 |

### 檔案操作
| 指令 | 用途 |
|------|------|
| `touch <檔名>` | 建立空檔案 |
| `mkdir <資料夾名>` | 建立資料夾 |
| `cp <來源> <目的>` | 複製檔案 |
| `mv <來源> <目的>` | 移動檔案（也可以改名） |
| `rm <檔名>` | 刪除檔案（無垃圾桶，不可復原！） |

### 其他
| 指令 | 用途 |
|------|------|
| `pwd` | 顯示目前所在目錄 |
| `cd <路徑>` | 切換目錄 |
| `head -N` | 只顯示前 N 行 |
| `echo ~` | 顯示 ~ 代表的路徑 |

## 重要觀念

### 路徑規則
- **絕對路徑**：`/` 開頭，從根目錄算起（如 `/home/johnny/practices/`）
- **相對路徑**：不加 `/`，從目前位置算起（如 `folder1/`）
- **`~`**：家目錄的縮寫，等於 `/home/johnny`，只有放在開頭才有效

### 常用旗標（flag）
- `-h`：human-readable，用 K、M、G 顯示大小
- `-s`：summarize，只顯示總計
- `-l`：long，顯示詳細資訊
- `-a`：all，包含隱藏檔（`.` 開頭的檔案）
- `-R`：recursive，遞迴列出子目錄

### 隱藏檔
- `.` 開頭的檔案是隱藏檔（如 `.bashrc`, `.config/`）
- `ls` 看不到，`ls -a` 才看得到

### Linux 重要目錄
| 目錄 | 用途 |
|------|------|
| `/home` | 使用者的家目錄 |
| `/etc` | 系統設定檔 |
| `/var` | 變動資料（log、快取） |
| `/tmp` | 暫存檔（重開機可能清除） |
| `/usr` | 使用者安裝的程式 |

### rm 的注意事項
- CLI 的 `rm` 沒有垃圾桶，刪了就沒了
- 刪除前先用 `ls` 確認目標
- 不確定時用 `rm -i` 逐一確認

### 讀錯誤訊息
- `是個目錄` → 你把目錄當指令了
- `沒有此一檔案或目錄` → 路徑打錯或不存在
- `拒絕不符權限的操作` → 你沒有權限（可能路徑也錯了）

## 今日踩過的坑
1. `ls /etc` 忘了打 `ls`，直接執行 `/etc`
2. `head 10` 忘了加 `-`，bash 以為 10 是檔案
3. `/tpm` 打成 `/tmp` 的錯字
4. `touch file1.txt touch file2.txt` → `touch` 變成了一個檔案名
5. `cp file1.txt /folder1` → `/folder1` 是絕對路徑，不是相對路徑
6. `/~/practices/` → `~` 不能跟 `/` 開頭混用
7. `pdw` → 應該是 `pwd`
